# Student-Management-system
A simple C++ project to manage students record like id, name, address, roll no, marks. 


This is a basic C++ program that demonstrates inheritance in Object-Oriented Programming (OOP).
A base class Person is used to store general information.
A derived class Student inherits from Person and stores educational details.
🎯 Features

Input and display personal information (ID, Name, Address)
Input and display student details (Roll Number, Marks)
Demonstrates inheritance concept
Beginner-friendly code

🏗️ Classes Used
🔹 Person (Base Class)

Contains:
id
name
address
Functions:
GetInfo() → Takes input from user
ShowInfo() → Displays person details

🔹 Student (Derived Class)

Inherits from Person
Contains:
roll
marks
Functions:
GetEdu() → Takes student academic data
ShowEdu() → Displays student academic data
